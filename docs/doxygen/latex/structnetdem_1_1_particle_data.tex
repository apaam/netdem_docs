\doxysubsection{Detailed Description}
A struct defining the particle data for MPI communication. 

The \doxylink{structnetdem_1_1_particle_data}{Particle\+Data} struct contains all properties of the \doxylink{classnetdem_1_1_particle}{Particle} class except for pointers or references, as these cannot be exchanged between processes due to addressing rules and memory space limitations.

For information on the meanings of the properties, please refer to the \doxylink{classnetdem_1_1_particle}{Particle} class defined in \doxylink{particle_8hpp}{particle.\+hpp}.
