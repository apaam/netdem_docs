\doxysubsection{Detailed Description}
To dump particle data into vtk files. This is a post-\/modifier, which will be excuted at the end of a DEM cycle.
\begin{DoxyItemize}
\item file\+\_\+dir\+: the directory of the output file
\item file\+\_\+basename\+: the name prefix of the outpuf file
\item data\+\_\+type\+: currently, vtk or dump (which is used in LAMMPS) format is implemented
\item save\+\_\+by\+\_\+cycles\+: data is saved in every \mbox{[}cycle\+\_\+interval\mbox{]} cycles if this flag is true. Otherwise, data is saved in every \mbox{[}time\+\_\+interval\mbox{]} mechanical time (i.\+e., time of the simulated world).
\item cycle\+\_\+previous\+: the id of the cycle in which the data is saved. If the difference between cycle\+\_\+previsous and current cycle id is greater than cycle\+\_\+interval. Data would be saved.
\item time\+\_\+previous\+: function similarly to cycle\+\_\+previous. 
\end{DoxyItemize}
