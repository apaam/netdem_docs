\doxysubsection{Detailed Description}
A class representing a poly superellipsoid with two different orders and three different axes. 

This class provides methods for setting and updating the position of the poly superellipsoid, computing its surface properties, generating STL models for visualization, and calculating signed distances and surface points.

Additionally, the \mbox{\hyperlink{classnetdem_1_1_poly_super_ellipsoid}{Poly\+Super\+Ellipsoid}} class is a subclass of the \mbox{\hyperlink{classnetdem_1_1_shape}{Shape}} class, which allows it to be used in combination with other shapes in compound objects.

The poly superellipsoid is defined by the following equation\+: ( ( x / axis\+\_\+a )$^\wedge$( 2 / order\+\_\+ab ) + ( y / axis\+\_\+b )$^\wedge$( 2 / order\+\_\+ab ) ) $^\wedge$ ( order\+\_\+ab / oder\+\_\+c ) + ( z / axis\+\_\+c )$^\wedge$( 2 / order\+\_\+c ) = 1 m and p indicate minus and plus to guarantee convexity, order should be in (0, 2)
