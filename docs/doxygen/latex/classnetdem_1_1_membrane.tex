\doxysubsection{Detailed Description}
A class that simulates a membrane. 

The {\ttfamily \mbox{\hyperlink{classnetdem_1_1_membrane}{Membrane}}} class is responsible for simulating a membrane. It provides functionality to initialize the membrane, set up boundary conditions, remesh the membrane, and solve the simulation using a lumped mass approach.
