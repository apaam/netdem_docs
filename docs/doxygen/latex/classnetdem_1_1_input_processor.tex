\doxysubsection{Detailed Description}
A class that processes input JSON files to modify a simulation. 

Create or modify a simulation using json input file. 

The {\ttfamily \mbox{\hyperlink{classnetdem_1_1_input_processor}{Input\+Processor}}} class is responsible for processing input JSON files and modifying a simulation accordingly. It provides functionality to initialize the class with a simulation object, process a JSON file, and process JSON data.
