\doxysubsection{Detailed Description}
A class for representing a level set function as a \mbox{\hyperlink{classnetdem_1_1_shape}{Shape}} object. 

This class provides methods for initializing a level set function from an STL file or distance map, computing signed distances and surface points, and generating STL models for visualization.

Additionally, the \mbox{\hyperlink{classnetdem_1_1_level_set}{Level\+Set}} class is a subclass of the \mbox{\hyperlink{classnetdem_1_1_shape}{Shape}} class, which allows it to be used in combination with other shapes in compound objects.
