\doxysubsection{Detailed Description}
A class representing the discretization of a peridynamic simulation. 

The \mbox{\hyperlink{classnetdem_1_1_peri_digm_discretization}{Peri\+Digm\+Discretization}} class represents the discretization of a peridynamic simulation. It includes methods for initializing the discretization from various data sources, manipulating the node positions and volumes, and writing the resulting node information to file.
