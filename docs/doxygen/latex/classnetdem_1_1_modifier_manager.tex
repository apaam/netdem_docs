\doxysubsection{Detailed Description}
Manages the modifiers in a simulation. Note that an modifier would take effects only if it is activated (i.\+e., it is inserted into pre\+\_\+modifier\+\_\+list or post\+\_\+modifier\+\_\+list).
\begin{DoxyItemize}
\item modifier\+\_\+lib\+: stores the list of defined modifiers.
\item modifier\+\_\+list\+: stores the list of activated modifiers.
\item scene\+\_\+state\+\_\+subscribers\+: stores the list of modifiers, whose properties need to update if the scene changes. For example, a body force modifier is defined for a certain particle. If this particle get out of the domain (i.\+e., the scene does not contain this particle any more), this body force modifier needs to be updated so that no more particles is in the particle\+\_\+list. 
\end{DoxyItemize}
