\doxysubsection{Detailed Description}
A class representing a cell in a simulation grid. 

cell is used for broad-\/phase contact detection. The particles in a sub-\/domain will first be sorted into cells, so that only the particles that fall in the same cell could be potentially contact with each other.
\begin{DoxyItemize}
\item bound\+\_\+min, bound\+\_\+max\+: coordinates of the lower and upper boundary of the rectangle cell.
\item particle\+\_\+list\+: particles that fall in this cell.
\item wall\+\_\+list\+: walls that fall in this cell.
\end{DoxyItemize}

This class keeps track of particles and walls that are located within its boundaries.
