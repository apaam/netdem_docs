\doxysubsection{Detailed Description}
A class for defining MPI datatypes for various data structures. 

The \mbox{\hyperlink{classnetdem_1_1_m_p_i_data_define}{MPIData\+Define}} class provides methods for defining MPI datatypes for \mbox{\hyperlink{structnetdem_1_1_particle_data}{Particle\+Data}}, \mbox{\hyperlink{structnetdem_1_1_bond_entry_data}{Bond\+Entry\+Data}}, \mbox{\hyperlink{structnetdem_1_1_collision_entry_data}{Collision\+Entry\+Data}}, \mbox{\hyperlink{structnetdem_1_1_contact_p_p_data}{Contact\+PPData}}, and \mbox{\hyperlink{structnetdem_1_1_contact_p_w_data}{Contact\+PWData}} structs. These datatypes allow the data structures to be communicated between MPI processes in the Peridigm code.
