Detailed Description

  • my_rank: the rank id of the self-process in a mpi run.
  • bound_min[3], bound_max[3] the lower and upper bounds of the x, y, and z coordinates of the whole domain.
  • vertices: the coordinates of the eight vertices of the domain (the domain is assumed to be a rectangle).
  • boundary_type: could be fixed, moving and periodic, which defines the behavior when a particle exceed the domain boundary.
