Detailed Description

a pool for storing and obtaining instances of particles and contacts. As particles and contacts are frequently added to or removed from the scene. The pool strategy is used to avoid the frequently construction and de-construction of object instances. When a particle or wall needs to be added, an instances will be obtained from the pool. When a particle or wall needs to be removed, it is recycled and stored in the pool.
