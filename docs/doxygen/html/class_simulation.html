Detailed Description

A DEM simulation contains six main moduli:

  • mpi_data_manager: manage the calculations involving data exchange between sub-domains
  • domain_manager: manage ththe calculations involving domain and sub-domains
  • scene: contains and manages the basic DEM objects (e.g., shapes, particle, and wall) for a DEM simulation
  • evaluator_manager: manage all the add-on features (i.e., those customized evaluations that are usually not hard coded in a DEM calculation cycle) for a DEM simulation
  • dem_solver: implments DEM algorithms and using DEM to solve the scene
  • command_processor: create or modify a DEM simulation based on commands defined by json file
  • mech_time: the mechanical time in the simulation world
  • mech_cycles: the mechanical cycles. Each cycle accouts for a [timestep] time incretement of the simulation mechanical time.
