\doxysubsection{Detailed Description}
A class representing a fragment in a discrete element method simulation. 

A namespace for classes related to network demography.

A namespace containing classes and functions for the netdem simulation.

A class that represents a boundary condition for Peridigm.

A class that represents a Peridigm block.

A domain splitter that uses a level set function to define the domain.

An abstract base class for domain splitters used in discrete element method simulations.

Namespace for the networked demolition simulation.

This class contains properties and methods for defining and manipulating DEM fragments, which can be used to model particles in simulations of granular materials and other systems.

This class defines an interface for splitting a domain, represented by an STL model, into discrete elements that can be used in a DEM simulation. Classes derived from this class can implement specific algorithms for splitting domains according to different criteria.

This class extends the \mbox{\hyperlink{classnetdem_1_1_domain_splittor}{Domain\+Splittor}} and \mbox{\hyperlink{classnetdem_1_1_level_set_function}{Level\+Set\+Function}} classes to implement a domain splitter that uses a level set function to define the geometry of the domain. The level set function is a scalar field that represents the distance between points in the domain and the boundary of the domain.

This class represents a Peridigm block, which is a group of nodes that share the same material properties and damage model. The nodes are identified by their indices in the discretized domain.

This class represents a boundary condition for Peridigm, which can be either a prescribed displacement or a body force. It defines the set of nodes that are subject to the boundary condition, as well as the type and magnitude of the applied load. The boundary condition can also be defined as time-\/dependent or not.

Namespace for the network dem package.
