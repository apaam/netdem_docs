\doxysubsection{Detailed Description}
A class that manages the domain and sub-\/domains for a DEM simulation. 

The {\ttfamily \mbox{\hyperlink{classnetdem_1_1_domain_manager}{Domain\+Manager}}} class is responsible for managing the domain and sub-\/domains in a DEM simulation. It provides functionality to initialize the domain and sub-\/domains, set their boundaries and decomposition, and retrieve ghost sub-\/domains for MPI data exchange.
