\doxysubsection{Detailed Description}
An interface class used to modify a DEM simulation. 

The \mbox{\hyperlink{classnetdem_1_1_modifier}{Modifier}} class allows for the addition of various features to a DEM simulation, such as the effects of gravity, data dumping, and scene rendering.
