\doxysubsection{Detailed Description}
A class representing a poly superquadric with three different axes and orders. 

This class provides methods for setting and updating the position of the \mbox{\hyperlink{classnetdem_1_1_poly_super_quadrics}{Poly\+Super\+Quadrics}} object, computing its surface properties, generating STL models for visualization, and calculating signed distances and surface points.

Additionally, the \mbox{\hyperlink{classnetdem_1_1_poly_super_quadrics}{Poly\+Super\+Quadrics}} class is a subclass of the \mbox{\hyperlink{classnetdem_1_1_shape}{Shape}} class, which allows it to be used in combination with other shapes in compound objects.

The poly superquadric is defined by the following equation\+:

( x / axis\+\_\+a )$^\wedge$( 2 / order\+\_\+a ) + ( y / axis\+\_\+b )$^\wedge$( 2 / order\+\_\+b ) + ( z / axis\+\_\+c )$^\wedge$( 2 / order\+\_\+c ) = 1

where m and p indicate minus and plus to guarantee convexity. The order should be in (0, 2).
