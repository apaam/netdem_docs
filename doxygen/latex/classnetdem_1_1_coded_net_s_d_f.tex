\doxysubsection{Detailed Description}
A class for representing a shape using a neural network-\/generated signed distance function and a latent code. 

This class provides methods for loading and initializing a neural network-\/generated signed distance function and a corresponding latent code, as well as computing signed distances and surface points from a given position. \begin{Desc}
