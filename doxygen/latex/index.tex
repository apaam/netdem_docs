 \href{https://apaam.github.io/netdem_docs/}{\texttt{ {\bfseries{Net\+DEM}}}} is a C++ program library targeted at the computational mechanics of irregular granular materials and utilizing machine learning tools to boost the computational effciency.

It is currently capable of performing basic and general DEM simulations, with following features\+:


\begin{DoxyItemize}
\item Sphere and triangle facets contact solver
\item GJK contact solver for convex particles
\item SDF contact solver for arbitrary (convex and concave) particles
\item Hybrid Open\+MP and MPI parallel computing
\item Integrated \href{https://www.mlpack.org/}{\texttt{ mlpack}} machine learning environment
\end{DoxyItemize}

The supported particle shapes include {\ttfamily sphere}, {\ttfamily cylinder}, {\ttfamily poly-\/super-\/ellipsoid}, {\ttfamily poly-\/super-\/quadrics}, {\ttfamily spherical harmonics}, {\ttfamily triangle mesh}, {\ttfamily level set}, etc.\hypertarget{index_autotoc_md1}{}\doxysection{Get started}\label{index_autotoc_md1}
