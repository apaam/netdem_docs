Detailed Description

Manages the evaluators in a simulation. Note that an evaluator would take effects only if it is activated (i.e., it is inserted into pre_evaluator_list or post_evaluator_list).

  • evaluator_lib: stores the list of defined evaluators.
  • evaluator_list: stores the list of activated evaluators.
  • scene_state_subscribers: stores the list of evaluators, whose properties need to update if the scene changes. For example, a body force evaluator is defined for a certain particle. If this particle get out of the domain (i.e., the scene does not contain this particle any more), this body force evaluator needs to be updated so that no more particles is in the particle_list.
