Detailed Description

A class representing a deformable particle simulated using the Finite Element Method (FEM).

This class inherits from the Particle class and includes additional functionality for simulating deformable particles using FEM. It contains a triangular mesh shape and a tetrahedral mesh used for the simulation, as well as functions to update the motion and shape of the particle, apply contact forces, and save data in VTK format.
