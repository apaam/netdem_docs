Detailed Description

A class that represents a feedforward neural network for regression.

The RegressionNet class is a specific implementation of a feedforward neural network for regression. It provides functionality to reset the model, add layers to the model, train the model with data, predict with the model, calculate the gradient of the model with respect to the input data, and load/save the model to disk.
