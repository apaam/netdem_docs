enum ContactAlgorithmType
strong

Three types of contact detection and resolution algorithm.

  • geometric: contact is resolved based on geometric algorithms. If this type of algorithm is choosed. Each pair of shapes (e.g., sphere_sphere, sphere_plane, sphere_oval, oval_oval, etc) must have its own contact detection and resolution algorithm.
  • GJK: contact is resolved based on the GJK algorithm. This is a general algorithm that can be used for arbitary-shaped convec particles.
  • machine_learning: using machine learning to detect and resolve contacts. A machine learning model would take particle size, shape descriptors as inputs, and outputs true if particles are in contact. In addition, if particles are in contact, another machine learning model is used to resolve the contact geometric features.
