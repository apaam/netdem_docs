Detailed Description

Class cell is used for broad-phase contact detection. The particles in a sub-domain will first be sorted into cells, so that only the particles that fall in the same cell could be potentially contact with each other.

  • bound_min[3], bound_max[3]: coordinates of the lower and upper boundary of the rectangle cell.
  • vertices[8][3]: vectices of the rectangle cell.
  • particle_list: particles that fall in this cell.
  • wall_list: walls that fall in this cell.
