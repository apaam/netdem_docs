Detailed Description

( ( x / axis_a )^( 2 / order_ab ) + ( y / axis_b )^( 2 / order_ab ) ) ^ ( order_ab / oder_c ) + ( z / axis_c )^( 2 / order_c ) = 1 m and p indicate minus and plus to guarantee convexity, order should be in (0, 2)
